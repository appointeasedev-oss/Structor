# Structor

Headless Console UI for Structor.

## Installation

To run Structor on your Windows CMD:

1.  **Install Python:** Make sure you have Python 3.8+ installed. You can download it from [python.org](https://www.python.org/).
2.  **Clone the Repository:**
    ```cmd
    git clone https://github.com/appointeasedev-oss/Structor.git
    cd Structor
    ```
3.  **Install Dependencies:**
    ```cmd
    pip install -r requirements.txt
    ```
4.  **Run Structor:**
    ```cmd
    python main.py
    ```

## Features

-   **ASCII Art Splash Screen:** Professional terminal branding.
-   **Terms and Conditions:** Secure onboarding flow.
-   **Command Dropdown:** Interactive selection for all functions.
-   **HeHo API Integration:** Set and manage your HeHo environment variables.
-   **AI Provider Management:** Support for OpenRouter and Puter.
-   **Open Interpreter:** Integrated AI model runner with custom base prompts.
